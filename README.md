Daulse Portfolio

A simple **personal portfolio website** built with HTML and CSS to showcase your work, skills, and presence online. Designed to be clean, responsive, and easy to customize.

Live Preview

If deployed via GitHub Pages or your own hosting, this portfolio serves as your **online resume** and professional showcase.

Structure

daulse_portfolio/
├── dadabe.jpg       # Your portrait or hero image
├── index.html       # Main webpage
├── style.css        # Site styling
└── README.md        # Project documentation


Features

* Responsive layout compatible with desktop and mobile
* Clean sections to present about me, skills, projects, contact info
* Simple HTML and CSS — no framework needed
* Easy to edit and extend for your own portfolio


How to Use

1. Clone the repository

   git clone https://github.com/kasiincognito/daulse_portfolio.git

2. index.html in your browser to view the site.

3. **Customize your details**

   * Replace text content in `index.html`: your name, bio, education, project links
   * Replace `dadabe.jpg` with your own image
   * Edit styles in `style.css` to change colors, fonts, layout


Deployment

You can host the portfolio easily:

GitHub Pages

1. Go to your repo’s Settings → Pages
2. Choose the `main` branch and `/ (root)` folder
3. Your portfolio will be published at:

   https://<your-username>.github.io/daulse_portfolio
